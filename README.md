# 실시간 자막 (Voice Subtitle)

말을 하면 **큰 글씨**로 보여 주는 실시간 자막 웹앱입니다.
구글의 ‘실시간 자막’처럼 화면 아래에서 글이 천천히 올라옵니다.

## 특징
- 🆓 **무료 · 온디바이스** — 안드로이드 크롬에 내장된 음성 인식(Web Speech API)을 사용. API 키·요금·설치가 필요 없습니다.
- 👵 **노인 친화** — 굵고 큰 글씨, 높은 대비(검은 배경·흰 글씨), 큰 버튼.
- 🐢 **읽기 편함** — 글이 한 줄씩 부드럽게 나타나고, 아직 확정 안 된 말은 흐린 색으로 보여 줍니다.
- 📱 **앱처럼 설치** — 홈 화면에 추가하면 앱 아이콘으로 실행됩니다(PWA).
- 🔋 화면 자동 꺼짐 방지(지원 기기).

## 사용 방법 (안드로이드)
1. **크롬(Chrome)** 브라우저로 이 앱 주소를 엽니다.
   > ⚠️ 마이크는 보안상 **https 주소** 또는 `localhost` 에서만 동작합니다.
2. **‘🎙️ 자막 시작’** 버튼을 누르고, 마이크 권한을 **허용**합니다.
3. 말을 하면 큰 글씨로 자막이 나타납니다.
4. ⚙️ 버튼에서 **글자 크기**를 바꿀 수 있습니다(작게/보통/크게/아주 크게).
5. 🗑️ 버튼으로 화면을 지웁니다.

### 홈 화면에 앱으로 추가
크롬 메뉴(⋮) → **‘홈 화면에 추가’** → 아이콘으로 실행.

## 호스팅 방법 (https 필요)
가장 쉬운 무료 방법:

- **GitHub Pages**: 이 폴더를 깃허브 저장소에 올리고 Pages 활성화 → `https://아이디.github.io/저장소/`
- **Netlify / Vercel**: 폴더를 드래그&드롭으로 배포.

### 로컬에서 테스트 (PC)
```powershell
# 이 폴더에서
python -m http.server 8000
```
PC 크롬에서 `http://localhost:8000` 접속.
> 휴대폰에서 같은 와이파이로 PC IP에 접속하면 http라서 마이크가 막힐 수 있습니다. 휴대폰 테스트는 위 https 호스팅을 권장합니다.

## 파일
| 파일 | 설명 |
|------|------|
| `index.html` | 앱 본체 (화면 + 음성 인식 로직) |
| `manifest.json` | PWA 설치 정보 |
| `sw.js` | 오프라인 캐시 (서비스 워커) |
| `icon.svg` | 앱 아이콘 |

## 알아두기
- 음성 인식 정확도와 ‘온디바이스 여부’는 안드로이드 버전·기기에 따라 다릅니다. 최신 안드로이드 크롬은 단말 내 인식을 사용합니다.
- 인식 엔진이 잠깐 멈추면 자동으로 다시 시작합니다(연속 자막 유지).
- iOS 사파리는 Web Speech API 지원이 제한적이라 안드로이드 크롬을 권장합니다.
